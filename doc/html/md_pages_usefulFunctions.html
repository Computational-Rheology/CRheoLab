///We need to compute the projection of a vectorField to determine the component of a vector in a given direction.

The algorithm for computing a vector projection is based on the following steps;

  1. We create a function of dataType double, call it a function name mag in this case and assign to it parameters of classtype vector3 and object variable $ \vec {v}1 $ .
  2. Create an object of class type vectorField, call it a name result and assign to it a member function that returns the size of the ScalarField
  3. Create a variable (counter) $ i $ of type unsigned integer
  4. Set a condition for a loop to run until counter $ i $ is less than the magnitude of the scalarField and also increment counter $ i $ by 1

  5. Compute the product of the scalar $Scalar\lceil i\rfloor + \vec{z}\times Scalar\lceil i \rfloor $ for each cell and the coordinates $ (\vec{x},\vec{y},\vec{z}) $ of the given vector using the equation;

    $ \vec {v}1\times Scalar\lceil i \rfloor = \vec{x}\times Scalar\lceil i \rfloor + \vec{y}\times Scalar\lceil i \rfloor + \vec{z}\times Scalar\lceil i \rfloor $

  6. Assign the product to the created result variable to store value for the new vectorField

  7. Create a new object of class type scalarField, call it a name result and assign to it a member

    function that returns the size of the vectorField

  8. Create a counter (variable) of type unsigned integer, set the condition for a loop to run till

    counter $ i $ is less than the magnitude of the vectorField, also increment counter $ i $ by 1

  9. Compute the dot product of vector coordinates $ (\vec{x}_1,\vec{y}_1,\vec{z}_1) $ for each

    cell and given $ (\vec{x},\vec{y},\vec{z}) $ using the following equation;

    $ \vec{v}_1\lceil i \rfloor {\cdot}\vec{v} = \frac{\vec{x}_1\cdot\vec{x}}{\parallel x\parallel}+\frac{\vec{y}_1\cdot\vec{y}}{\parallel y\parallel}+\frac{\vec{z}_1\cdot\vec{z}}{\parallel z\parallel} $

  10. Then assign the computed dot product to the result variable to store the value for the new ScalarField.

/// We need to compute the magnitude of a vectorField

The algorithm for computing the magnitude of a vectorField is based on the following steps;

  1. We create a function of datatype double, call it a function name magVector in this case and assign

    to it parameters of classType vectorField and object variable vector $ \vec {v}1 $

  2. Create a variable (counter) $ i $ of data type unsigned integer

  3. Set a condition for a loop to run until counter $i$ is less than the maximuim size of the

    vectorField, also increment counter $i$ by 1

  4. Compute the square of the vector coordinates $ (\vec{x},\vec{y},\vec{z}) $ for each cell using the following equation;

    $ \vec{v}_1\cdot\vec{v}_1 = \vec{x}_1\times\vec{x}_1 + \vec{y}_1\times\vec{y}_1+\vec{z}_1\times\vec{z}_1$

  5. Create a new variable of name Product to store the value of the square product of the vector

    $ \vec {v}1 $

  6. Compute the square root of the Product using the following equation;

    $ {\sqrt{\vec{v}_1\times\vec{v}_1}} = \sqrt{Product}$

  7. Finally, create another variable named result of data type double to store value of the square root of the Product variable.
